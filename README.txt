Overview
=================================================================
This repository was cloned and reimplemented with c++11 from

	https://github.com/RaoofJM/nodejs-typescript-realworld-backend.git

Project is now imported into this repository with made refactoring changes
to follow its new design and structure for cloudflare deployments instead of nodejs.

Project goal
=================================================================
The aim is to teach Cloudflare Workers development and good coding convention
on web development for myself and for folks who are interested in it.
This is purely for learning and having fun with something that is useful
in web development.
